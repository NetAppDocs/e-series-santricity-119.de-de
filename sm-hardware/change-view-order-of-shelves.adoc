= Ändern Sie die Ansichtsreihenfolge der Regale im SANtricity System Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Reihenfolge der auf der Seite Hardware angezeigten Regalböden so ändern, dass sie der physischen Reihenfolge der Regalböden in einem Schrank entspricht.

.Schritte
. Wählen Sie *Hardware* aus.
. Wählen Sie oben rechts in der Regalansicht die Pfeile nach oben oder unten aus, um die Reihenfolge der auf der Hardware-Seite angezeigten Regale neu anzuordnen.
