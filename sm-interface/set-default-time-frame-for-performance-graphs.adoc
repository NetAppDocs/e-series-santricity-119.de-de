= Legen Sie den Standardzeitraum für Leistungsdiagramme im SANtricity System Manager fest.
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den standardmäßigen Zeitraum, der von den Leistungsdiagrammen angezeigt wird, ändern.

.Über diese Aufgabe
Die auf der Startseite und der Leistungsseite angezeigten Leistungsdiagramme zeigen anfänglich einen Zeitraum von 1 Stunde an. Die Einstellungen werden im lokalen Speicher des Browsers gespeichert, sodass alle Benutzer ihre eigenen Einstellungen haben können.

.Schritte
. Menü auswählen:Preferences [Set preferences].
. Wählen Sie in der Dropdown-Liste entweder *5 Minuten*, *1 Stunde*, *8 Stunden*, *1 Tag* oder *7 Tage* aus und bestätigen Sie, dass Sie die Operation durchführen möchten.
