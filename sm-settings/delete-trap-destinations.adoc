= Trap-Ziele im SANtricity System Manager löschen
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können eine Trap-Zieladresse löschen, sodass der Ereignismonitor des Storage Arrays keine SNMP-Traps mehr an diese Adresse sendet.

.Schritte
. Menü auswählen: Settings[Alerts].
. Wählen Sie die Registerkarte *SNMP* aus.
+
Die Trap-Zieladressen erscheinen in der Tabelle.

. Wählen Sie ein Trap-Ziel aus und klicken Sie dann oben rechts auf der Seite auf *Löschen*.
. Bestätigen Sie den Vorgang und klicken Sie dann auf *Löschen*.
+
Die Zieladresse wird nicht mehr auf der Seite "Warnungen" angezeigt.



.Ergebnisse
Das gelöschte Trap-Ziel empfängt keine SNMP-Traps mehr vom Ereignismonitor des Storage-Arrays.
