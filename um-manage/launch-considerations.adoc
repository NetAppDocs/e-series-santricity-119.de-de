= Erfahren Sie, wie Sie auf den SANtricity System Manager über Unified Manager zugreifen.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie wählen ein oder mehrere Speicherarrays aus und verwenden die Option Starten, um SANtricity System Manager zu öffnen, wenn Sie Speicherarrays konfigurieren und verwalten möchten.

Der System Manager ist eine in die Controller integrierte Anwendung, die über einen Ethernet-Management-Port mit dem Netzwerk verbunden ist. Er umfasst alle Array-basierten Funktionen.

Um auf den System Manager zugreifen zu können, müssen Sie Folgendes haben:

* Eines der hier aufgeführten Array-Modelle: link:https://docs.netapp.com/us-en/e-series/getting-started/learn-hardware-concept.html["Hardware-Übersicht der E-Series"^]
* Eine Out-of-Band-Verbindung zu einem Netzwerkmanagement-Client mit einem Webbrowser.
