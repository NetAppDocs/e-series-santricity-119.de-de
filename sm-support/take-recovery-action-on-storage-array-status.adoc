= Ergreifen Sie Wiederherstellungsmaßnahmen für den Status des Speicherarrays im SANtricity System Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Maßnahmen bezüglich des Status Ihres aktuellen Storage Arrays ergreifen, indem Sie das Protokoll der nicht lesbaren Sektoren überprüfen, die Laufwerksports wieder aktivieren und den Wiederherstellungsmodus löschen.
