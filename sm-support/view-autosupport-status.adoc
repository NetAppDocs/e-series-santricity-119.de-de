= Status von AutoSupport im SANtricity System Manager anzeigen
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die AutoSupport-Seite zeigt Ihnen, ob die AutoSupport-Funktion und die einzelnen AutoSupport-Funktionen derzeit aktiviert sind.

.Schritte
. Menü auswählen: Support [Support Center > AutoSupport] Tab.
. Schauen Sie auf der rechten Seite direkt unterhalb der Registerkarten nach, ob die Basisfunktion AutoSupport aktiviert ist.
. Bewegen Sie den Mauszeiger über das Fragezeichen, um zu sehen, ob einzelne AutoSupport-Funktionen aktiviert sind.
