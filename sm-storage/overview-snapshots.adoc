Mit der Snapshot-Funktion können Sie zeitpunktgenaue Abbilder von Storage-Array-Volumes erstellen, die Sie für Backups oder Tests verwenden können.



== Was sind Snapshot-Images?

Ein Snapshot-Image ist eine logische Kopie von Volume-Daten, die zu einem bestimmten Zeitpunkt aufgenommen wurde. Wie ein Wiederherstellungspunkt ermöglicht ein Snapshot-Image das Zurücksetzen auf einen bekannten, guten Datensatz. Obwohl der Host auf das Snapshot-Image zugreifen kann, kann er es nicht direkt lesen oder darauf schreiben.

== Wie stelle ich Daten aus einem Snapshot wieder her?

Ein Rollback ist der Vorgang, bei dem Daten in einem Basis-Volume auf einen früheren Zeitpunkt zurückgesetzt werden. Sie können Snapshot-Daten über menu:Storage[Snapshots] zurücksetzen.
